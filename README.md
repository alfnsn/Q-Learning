# Q-Learning From Scratch

![Python](https://img.shields.io/badge/python-3.12-blue)

Repository ini berisi proses belajar dan implementasi Q-Learning dari dasar.

---

## About

Q-Learning adalah algoritma Reinforcement Learning yang memungkinkan agen belajar mengambil keputusan optimal melalui trial and error menggunakan reward.

Repo ini dibuat untuk:
- Memahami konsep dasar Reinforcement Learning
- Mengimplementasikan Q-Learning dari nol
- Menganalisis proses training dan konvergensi
- Membangun mini project berbasis Q-Learning

---

## Goals

- Memahami dasar Reinforcement Learning
- Mengimplementasikan Q-Learning dengan atau tanpa library RL
- Membangun mini project sederhana
